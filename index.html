This tool is designed to streamline the process of highlighting faculty research accomplishments and preparing documentation for accreditation purposes. It provides a complete record of citations done by an author, generates a summary, and displays a graph based on the analyzed records.

The tool accepts the following file formats: Excel spreadsheets (.xlsx), BibTeX files (.bib).
Important: You can upload one file at a time, and the file must contain at least the Author's Name.

Your input file should be structured as follows:

Author Name Institution Name
John Doe University of Example
Jane Smith Another University

Usage Instructions:
  1. Click on "Choose File" or drag and drop your file into the designated area.
  2. Select the file you wish to upload.
  3. Press the Upload button.

The generated summary is brief and contains key points about the author's publications. You can download the summary as a Word document.

Currently, the limitations are being assessed and will be updated in future releases.
